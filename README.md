# 🛒 Retail Sales Performance Dashboard

## 📊 Project Overview
Interactive Excel Dashboard analyzing 2,000 retail orders to uncover 
sales performance trends, top salesperson rankings, regional revenue 
breakdown, and product category analysis.

## 🎯 Key Insights
- 📈 Monthly revenue trend analysis across 2 years
- 🏆 Top 5 salespersons ranked by total revenue
- 🗺️ Revenue breakdown by Region (North, South, East, West)
- 🛍️ Product category performance analysis
- 👥 Customer segment analysis (Enterprise, Consumer, Small Business)

## 📌 KPIs Tracked
| Metric | Value |
|--------|-------|
| Total Revenue | ₹5.1Cr+ |
| Total Orders | 2,000 |
| Profit Margin | 25.34% |
| Avg Order Value | ₹25,500 |

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Custom Formatting
